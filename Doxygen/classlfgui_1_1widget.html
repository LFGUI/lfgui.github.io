template<typename T >
T* lfgui::widget::add_child ( T *  w)
inline

Adds a child widget. Returns a pointer to the widget. std::unique_ptr is used internally to handle the given widget, no need for a manual delete.

Example:

auto button=widget->add_child(new lfgui::button(50,155,100,25)); // add a button at 50,155 with the size 100,25
auto slider=widget->add_child(new lfgui::slider(50,95,180,50)); // add a slider
